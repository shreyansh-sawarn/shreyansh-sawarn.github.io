# shreyansh-sawarn.github.io

Personal portfolio of **Shreyansh Sawarn** — DevOps Engineer.

**Live:** [shreyansh-sawarn.github.io](https://shreyansh-sawarn.github.io/)

Cyberpunk × Japanese cherry-blossom design. Built deliberately with **three
files and zero dependencies** — no framework, no build step, instant loads.

## Features

- Dark/light theme with persistent toggle (sumi-e sakura artwork processed
  separately per theme)
- Falling sakura petals and katakana digital rain on `<canvas>`
- Typewriter role line with glitch effects
- Interactive CI/CD pipeline — press **▶ deploy** and watch it ship
- Animated impact-metric counters
- Live GitHub repo stats via the public API
- Terminal easter egg — press `` ` `` or the `>_` button and type `help`
  (↑/↓ history, Tab completion, and at least one hidden command)
- Themed 404 page, cycling favicons, JSON-LD, `llms.txt` for AI crawlers
- Responsive, keyboard-accessible, respects `prefers-reduced-motion`,
  Open Graph social previews

## Stack

HTML · CSS · vanilla JavaScript. That's the point.

## Run locally

```bash
git clone https://github.com/shreyansh-sawarn/shreyansh-sawarn.github.io.git
cd shreyansh-sawarn.github.io
# open index.html, or:
python -m http.server 8000
```

Deployed automatically via GitHub Pages from `main`.
